# Tour Package Panel

**Tour Package Panel** is a feature of **Admin** portal. Admin can create different packages by using this panel. This tor panel includes: `Tour Package Type` , `Accomodation Type`, `Accomodation Add`, `Room Type`, `Room Category`, `Meal type`, `Food Nmae`, `Transport Route`, `Transport Provider`, `Transport Mode`, `Transport Brands`

- Go to `Tour Package Panel` by clicking from the navigation bar.
