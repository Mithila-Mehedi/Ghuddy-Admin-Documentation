## Accommodations Add

- Click on `Accommodations Add` to go to the page.

- Now click on `Add New` button to add a new Accommodation. Accommodation could be any Hotel, Resort, Villa, Cottage name. For example, Hotel Arts, Dwipantor Beach Resort etc.

![Tour Package Panel](pages/assets/acc1.png)

- A popup modal will open after clicking on `Add New` button. Fill up the input field to add a new accommodation successfully. This modal contains: `Accommodations Nmae`, `Accommodation Address`, `Accommodation Type` and `Submit` button.

- User can find a dropdown menu in the `Accommodation Type` field which are added before in the Accommodation Type sub panel.

- Submit button will enable after filling up the input field.

![Tour Package Panel](pages/assets/acc2.png)

- New Add created successfully. Scroll down to find the newly added accommodation Add in the `All Accommodations Add` list

![Tour Package Panel](pages/assets/acc3.png)
