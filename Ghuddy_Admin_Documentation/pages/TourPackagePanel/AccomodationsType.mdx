## Accommodations Type

- Click on `Accommodations Type` to go to the page.

- Now click on `Add New` button to add a new Accommodations type. Accommodations type could be `Hotel`, `Resort`, `Villa`, `Cottage`.

![Tour Package Panel](pages/assets/acc1.png)

- A popup modal will open after clicking on `Add New` button. Fill up the input field to add a accommodations type successfully. This modal contains: `Accommodations Type Nmae` and `Submit` button.

- Submit button will enable after filling up the input field.

![Tour Package Panel](pages/assets/acc2.png)

- New type created successfully. Scroll down to find the newly added accommodation type in the `All Accommodations Type` list

![Tour Package Panel](pages/assets/acc3.png)
